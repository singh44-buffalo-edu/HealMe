HealMeDaily · Design System v1

Calm clinical, on paper.

A private, single-patient record. Warm paper neutrals, one evergreen accent, serif headings over a precise sans, and mono numerals for every measurement. Three data classes, always distinguishable: measured (ink), live device (evergreen pulse), AI-derived (iris, always with confidence). Themes over Mantine / @medplum/react.

Mantine theming notes

Maps to a Mantine theme object: primaryColor: evergreen, custom colors.paper / .iris / .status, fontFamily / headings.fontFamily / fontFamilyMonospace, radius 8–14, shadows nearly off. @medplum/react components inherit via MantineProvider; custom components use the same tokens.
